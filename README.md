# Sistema de Gestión de Tareas

Sistema web para organizar y gestionar tareas académicas de estudiantes.

## Problema
Muchos estudiantes olvidan sus tareas o las gestionan de manera informal, lo que genera desorganización y bajo rendimiento académico.

## Solución
Un sistema que permita registrar, visualizar, editar y marcar tareas como completadas.

## Funciones principales
- Crear tarea
- Editar tarea
- Marcar tarea como completada
- Eliminar tarea
